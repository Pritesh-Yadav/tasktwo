Learn More about GIT Commands

GIT Commands

git add for adding git files
git status for checking the status of the code
git commit for committing changes in the repository
git checkout for shifting on to other branches
git merge for merging two different files
git config for the authentication of the user
git remote show origin for showing cloned repository

Linux Commands

pwd command is used to tell where we are located in the file system
ls command is used to list all the files and directories
cd command is used to change the directory
dir command is used to list all the directories in the current working directory
mkdir command is used to create a new directory by the name

Vi Commands

vi"filename" command creates a file with FileName in the working directory
i command turns on the insert mode and edit the file
esc command exits the insert mode
w command writes the text in the file
